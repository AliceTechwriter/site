Чат

Для получения оперативной поддержки можно использовать специальный workspace в Slack.

Q&A

Для задания вопросов (поиска среди существующих) можно использовать соответствующую метку на Stack Overflow.

Управление задачами

Система управления задачами, в которой можно сообщать об ошибках, следить за выполнением задач, а также добавлять предложения по расширению функционала, находится внутри соответствующих репозитариев на GitHub.

Блог

Для чтения статей и другой информации о платформе можно использовать соответствующий хаб на Habr.com.

Новости

Для чтения новостей и событий о платформе можно использовать соответствующий канал в Twitter.

Исходники

Исходники платформы и плагина к IDEA находятся в соответствующих репозитариях на GitHub.

Система сборки

Следить за процессами автоматической сборки можно при помощи соответствующего веб-интерфейса в Jenkins.

Почта

По всем остальным вопросам можно обращаться по адресу info@lsfusion.org.
